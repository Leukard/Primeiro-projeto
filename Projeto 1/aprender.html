5 Passos para Aprender HTML

  1. Conhecer a estrutura básica

    Toda página HTML começa com <html>, <head> e <body>.

  2. Aprender as tags principais

    Algumas delas são <h1>, <p>, <a> e <img>.

  3. Criar listas e adicionar imagens

    Use <ol> ou <ul> para listas e <img> para imagens.

  4. Usar links

    Com a tag <a> você pode ligar uma página a outra.

  5. Praticar bastante

    Crie seus próprios exemplos para aprender de verdade.
